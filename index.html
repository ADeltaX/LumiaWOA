It's the traditional Windows, you know and love...

This project ports the Windows 10 Desktop operating system to the Lumia 950 and Lumia 950 XL. It is the same edition of Windows 10 you're used to on your traditional laptop or desktop computer, but it's the version for ARM64 (armv8a) processors. It can run ARM64, ARM and x86 applications (the later via emulation) just fine.

...with Continuum*

*Continuum can only be used via Miracast currently.

...but with a mobile twist.

Mobile Shell is a fully featured adaptive shell aiming to mimic the Windows 10 Mobile appearance. MobileShell brings back the navigation bar and the status bar, puts your notification toasts at the top, status icons at a glance, and activates only when the phone is in tablet mode.

Mobile Shell also supports landscape mode, and adjusts perfectly to the phone current state. Mobile Shell is made by @ADeltaX and is not included by default, but can be gotten manually from the link below.

Get Mobile Shell from the Microsoft Store View on Github
With call capabilities*

This project backports the Cellular stack from Windows 10 Mobile to Windows 10 Desktop. On supported versions of Windows, you can make calls, texts, browse the internet via cellular. WOA Dialer is an application allowing you to make and manage calls on your device. WOA Dialer is bundled with the project by default, along with the classic Microsoft Phone application.

View WOA Dialer on Github

* Cellular support is still unfinished and might be broken in some areas. Cellular Calls and Cellular Texts are supported up to Windows 10 Build 18363. Versions higher than build 18363 will only support Cellular Data. Dual SIM devices may have issues fetching the default Carrier APN settings, a provisioning package for APN may be required. The advanced settings page for Cellular in the Windows Settings app may crash. Experience may varies with Carriers and devices. This software stack has not been approved for use with emergency services. As a consequence it should not be used as your primary way of communication. VoLTE (IMS) stack while present is not functional.

...and text capabilities*

With the Chat application, you can send and get text messages. MMS messages remain unsupported as of now.
